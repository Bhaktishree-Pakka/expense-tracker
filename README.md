
# 💸 Expense Tracker

A modern, dark-themed Expense Tracker web application built using Next.js and React. This app helps users manage daily expenses efficiently with a clean UI and real-time insights.

---

## 🚀 Live Demo

👉https://expense-tracker-three-zeta-71.vercel.app/

---

## ✨ Features

- ➕ Add and delete expenses
- 🏷️ Categorize expenses (Food, Travel, Shopping, Other)
- 🔍 Search and filter functionality
- 💾 Persistent data using LocalStorage
- 📊 Real-time total calculation
- 🌙 Fully responsive dark-mode UI
- ⚡ Fast and smooth user experience

---

## 🛠️ Tech Stack

- Next.js
- React

- CSS

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/Bhaktishree-Pakka/expense-tracker.git
```

### 2. Navigate to the project folder

```
cd expense-tracker
```

### 3. Install dependencies

```
npm install
```

### 4. Run the development server

```
npm run dev

```
